How to Use

1. Select the start and end dates.

2. If you want to calculate time, check the "Do you want to calculate the time?" box and enter the start and end times.

3. If you want to calculate your salary, check the "Do you want to calculate your salary?" box and enter your salary.

4. Click the "Calculate Days" button to see the results.
